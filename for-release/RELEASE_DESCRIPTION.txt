## Life Calendar v1.0.0

Personal journal calendar with statistics and retrospectives for Obsidian.

### Features
- Calendar view for daily notes
- Streak tracking
- Statistics dashboard
- Periodic notes support (day/week/month/quarter/year)
- Retrospectives ("memory" module)
- Templater integration
- Russian language interface

### Installation
Download the files from this release and extract them to `.obsidian/plugins/life-calendar/` in your vault.
